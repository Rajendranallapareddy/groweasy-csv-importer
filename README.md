# GrowEasy AI‑Powered CSV Importer

A production‑ready SaaS application that uses AI to intelligently map arbitrary CSV columns to a standardized CRM format. Built with Next.js 15, Express, and OpenAI.

## Features

- Drag‑and‑drop CSV upload with instant preview
- Intelligent AI field mapping (supports any column names)
- Batch processing with retries
- Detailed import statistics (imported / skipped)
- Export results as JSON
- Fully responsive UI with dark mode
- Virtualized tables for large datasets
- Dockerized for easy deployment

## Tech Stack

- **Frontend**: Next.js 15, TypeScript, Tailwind CSS, shadcn/ui, TanStack Table, Framer Motion
- **Backend**: Node.js, Express, TypeScript, Multer, OpenAI SDK
- **Infrastructure**: Docker, Docker Compose
